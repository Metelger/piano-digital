Como utilizar as teclas?

  • Nota Dó: tecla 'a' -> Frequência (Hz): 261.63
  • Nota Ré: tecla 's' -> Frequência (Hz): 293.66
  • Nota Mi: tecla 'd' -> Frequência (Hz): 329.63
  • Nota Fá: tecla 'f' -> Frequência (Hz): 349.23
  • Nota Sol: tecla 'j' -> Frequência (Hz): 392.00
  • Nota Lá: tecla 'k' -> Frequência (Hz): 440.00
  • Nota Si: tecla 'l' -> Frequência (Hz): 493.88
  • Nota Dó: tecla 'ç' -> Frequência (Hz): 523,25
